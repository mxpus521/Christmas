# 理性圣诞 · 人文与秩序的节日叙事 
# Rational Christmas: A Narrative of Humanity and Order

这是一个基于 **纯前端技术栈** 构建的创意型圣诞主题网站。不同于传统的“童话与童趣”风格，本项目旨在从理性、人文与社会秩序的视角重新审视圣诞节，探讨其在现代文明中的深层意义。

## 🎄 项目定位 | Positioning

**核心理念：** 克制 / 冷静 / 知性 / 现代 
**目标：** 将节日从消费主义与低幼叙事中剥离，回归文明与共同体的本质。

## ✨ 核心特性 | Features

### 1. 深度叙事架构 (Narrative Architecture)
*   **Hero Section**: “Beyond the Myth” —— 视觉与文案的双重声明。
*   **认知拆解 (Deconstruction)**: 三大维度剖析节日的社会功能（超越消费、结构化叙事、时间锚点）。
*   **时间与秩序 (Entropy & Order)**: 交互式进度环，展现从“私域亲密”到“自然永恒”的逻辑递归。
*   **历史维度 (Historical Dimensions)**: 从起源到现代的文明演进时间轴。
*   **冬夜语录 (Winter Quotes)**: 中英双语理性语录，触达内心深处的静谧。

### 2. 视觉与交互 (Visuals & Interactions)
*   **Premium 设计**: 采用 `#0b0f14` 深空底色配合 `#cfa96b` 冷金点缀，营造高端智性视觉感。
*   **AI 生成视觉**: 包含由 AI 生成的极简主义几何圣诞树背景，象征秩序之美。
*   **动态粒子系统**: 自研 Canvas 雪花系统，模拟冬夜微光，克制而灵动。
*   **多语言支持**: 完善的 **ZH / EN** 双语切换逻辑。

## 🛠️ 技术实现 | Tech Stack

*   **HTML5**: 语义化结构，高度优化的 SEO 元数据。
*   **CSS3**: 采用 Flex/Grid 布局，变量化设计系统，响应式适配移动端。
*   **Vanilla JS**: 
    *   **Intersection Observer API**: 实现滚动触发的流式动效。
    *   **Canvas API**: 实现高性能粒子系统。
    *   **Data-Attribute Logic**: 多语言交互的数据解耦。
